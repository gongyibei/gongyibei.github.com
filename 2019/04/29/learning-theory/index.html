包裹式（wrapper model feature selection）

常用的有前向搜索算法、后向搜索算法等，前向搜索与后向搜索均属于启发式搜索，不能保证找到最好的特征子集合

前向搜索算法

  1. 初始化特征集合 $\mathcal{F}=\varnothing$

  2. Repeat {

    (a) 现有特征的集合加上剩下的$k$个特征中的一个可以得到一个模型，由此得到的$k$个模型构成模型集合$\mathcal{H}$（$|\mathcal{H}|=k$）。

    (b) 用特征选择的方法从模型集合中选择出最合适的模型，产生该模型所加入特征即为本次要加入特征。将特征加入特征集合$\mathcal{F}$。

    }

后向搜索算法

  1. 初始化特征集合 $\mathcal{F}= \text{所有特征}$

  2. Repeat {

    (a) 现有特征的集合的$k$个特征里删除一个特征可以得到一个模型，由此得到的$k$个模型构成模型集合$\mathcal{H}$（$|\mathcal{H}|=k$）。

    (b) 用特征选择的方法从模型集合中选择出最合适的模型，产生该模型所删除的特征即为本次所要删除的特征。将该特征从特征集合$\mathcal{F}$中删除。

    }

过滤式（filter feature selection）

算法步骤

  1. 求出每个特征$x_i$与y的相关性
  2. 按相关性的大小对特征进行排序
  3. 选择前k个特征，或者用交叉验证来决定选择前几个特征

两种方法比较

特点
wrapper model feature selection 效果好，运算量大
filter feature selection 运算量小

MAP估计（贝叶斯学派）

高斯环境下的MAP估计等价于L2正则化最小二乘，可以使模型更光滑。这种光滑特性依赖与我们所作的高斯假设。如果我们假设的先验分布大多数概率都接近与零就会有这样的平滑效果， 如拉普拉斯分布

我们希望给定数据集合$S$，对于某一个变量$x_i$,可以求得所$y_i$的概率分布

通过概率分布我们就可以求得其期望

然后我们需要知道给定数据集$S$下参数$\theta$的概率分布,可以通过贝叶斯公式计算

但是上面关于$\theta$的积分求解是非常困难的，我们可以用概率最大的$\theta_{\text{MAP}}$对应的模型近似代替上面的期望值

$\hat\theta$即为$p(\theta|S)$的极大值点,即MAP估计量
