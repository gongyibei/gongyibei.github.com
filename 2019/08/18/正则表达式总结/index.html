表示集合

Python Vim 描述
[abc] [abc] 匹配方括号中的任意一个字符，可用-表示字符范围。如[a-z0-9]匹配小写字母
abc abc 匹配除方括号中字符之外的任意字
\w \w 匹配字母数字及下划线
\W \W 匹配非字母数字及下划线
\s \s 匹配任意空白字符，等价于 [\t\n\r\f].
\S \S 匹配任意非空字符
\d \d 匹配任意数字，等价于 [0-9].
\D \D 匹配任意非数字

[abc]
在括号内(){}等特殊字符都回归正常但是[]要用加反斜杠|

表示次数

Python Vim 描述
* * 任意次数
+ \+ 1个或多个
? \? 任意个
{n} \{n} n个
{n,} \{n,} 至少n个
{n,m} \{n,m} n个到m个

对匹配的边界进行限制限制

格式 描述
^ 行首
$ 行尾
(?=re) 从该位置起右边的字符需要满足的
(?!re) 同上，但不满足
(? <=re) 该位置左边的字符需要满足的
(?<!re) 同上，但不满足
\A 匹配字符串开始
\Z 匹配字符串结束，如果是存在换行，只匹配到换行前的结束字符串。
\z 匹配字符串结束
\G 匹配最后匹配完成的位置。
\b 匹配一个单词边界，也就是指单词和空格间的位置。例如， ‘er\b’ 可以匹配”never” 中的 ‘er’，但不能匹配 “verb” 中的 ‘er’。
\B 匹配非单词边界。’er\B’ 能匹配 “verb” 中的 ‘er’，但不能匹配 “never” 中的 ‘er’。
