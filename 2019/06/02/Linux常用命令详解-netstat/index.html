字段介绍

  • Recv-Q ： 接受队列
  • Send-Q ： 发送队列
  • Local Address： 本地ip:port
  • Foreign Address ： 远程主机ip:port
  • State ：连接所处的状态
  • User ： 用户
  • Inode ： Inode
